MoonBit API Documentation

Введение

API MoonBit предоставляет доступ к данным о цене биткоина, фазах луны, астрологических событиях и их потенциальных корреляциях. Все запросы выполняются через HTTP и возвращают данные в формате JSON.

Базовый URL: http://localhost:5000/api

Аутентификация

В текущей версии API не требует аутентификации.

Bitcoin API

GET /api/bitcoin/current

Получить текущую цену биткоина

Возвращает текущую цену биткоина в указанной валюте.

Параметры запроса

Параметр Тип Обязательный Описание
currency string Нет Валюта (usd, eur, rub). По умолчанию: usd

Пример ответа

{
  "price": 55123.45,
  "currency": "usd",
  "last_updated": "2023-05-18T12:34:56Z",
  "change_24h": 1250.75,
  "change_percentage_24h": 2.32
}
GET /api/bitcoin/history

Получить историю цен биткоина

Возвращает историю цен биткоина за указанный период.

Параметры запроса

Параметр Тип Обязательный Описание
days integer Нет Количество дней (от 1 до 365). По умолчанию: 30
currency string Нет Валюта (usd, eur, rub). По умолчанию: usd

Moon API

GET /api/moon/current

Получить текущую фазу луны

Возвращает информацию о текущей фазе луны.

Пример ответа

{
  "date": "2023-05-18T12:34:56Z",
  "phase": 0.35,
  "phaseName": "Растущая луна (горб)"
}
GET /api/moon/period

Получить фазы луны за период

Возвращает информацию о фазах луны за указанный период.

Параметры запроса

Параметр Тип Обязательный Описание
startDate string Да Начальная дата в формате YYYY-MM-DD
endDate string Да Конечная дата в формате YYYY-MM-DD
GET /api/moon/next

Получить следующие значимые фазы луны

Возвращает информацию о следующих значимых фазах луны (новолуние и полнолуние).

Параметры запроса

Параметр Тип Обязательный Описание
count integer Нет Количество фаз (от 1 до 10). По умолчанию: 4

Пример ответа

[
  {
    "date": "2023-05-20T12:34:56Z",
    "type": "Новолуние",
    "phase": 0
  },
  {
    "date": "2023-06-04T12:34:56Z",
    "type": "Полнолуние",
    "phase": 0.5
  }
]

Astro API

GET /api/astro/events

Получить астрологические события

Возвращает список астрологических событий за указанный период.

Параметры запроса

Параметр Тип Обязательный Описание
startDate string Нет Начальная дата в формате YYYY-MM-DD. По умолчанию: текущая дата
endDate string Нет Конечная дата в формате YYYY-MM-DD. По умолчанию: +30 дней от startDate

Events API

GET /api/events/recent

Получить недавние события

Возвращает список последних событий, связанных с биткоином.

Параметры запроса

Параметр Тип Обязательный Описание
limit integer Нет Количество событий (от 1 до 100). По умолчанию: 5

Коды ошибок

Код Описание
400 Некорректный запрос. Проверьте параметры запроса.
404 Ресурс не найден.
500 Внутренняя ошибка сервера.
